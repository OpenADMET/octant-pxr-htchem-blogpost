## Minimal Hover Support

If the blog page already loads the OpenADMET/Quarto citation-tooltip behavior,
the anchors above should match that pattern. If it does not, this lightweight CSS
fallback gives hover text through the `data-tooltip` attribute:

```html
<style>
.cite-tip {
  position: relative;
  font-weight: 600;
  text-decoration: none;
}
.cite-tip:hover::after,
.cite-tip:focus::after {
  content: attr(data-tooltip);
  position: absolute;
  left: 50%;
  bottom: 1.6em;
  transform: translateX(-50%);
  z-index: 50;
  width: max-content;
  max-width: min(32rem, 80vw);
  padding: 0.55rem 0.7rem;
  border: 1px solid #d0d7de;
  border-radius: 6px;
  background: #ffffff;
  color: #24292f;
  box-shadow: 0 8px 24px rgba(140, 149, 159, 0.25);
  font-size: 0.86rem;
  line-height: 1.35;
  white-space: normal;
}
</style>
```
